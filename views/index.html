Library API Documentation

Welcome to the Bored API by the App Brewery.

Get a Book

Use this endpoint to retrieve a list of available books.

Example Request:

GET https://library-api-nestjs.vercel.app/book

Example Response:

[
    {
        "_id": "book-id",
        "title": "Next Js",
        "description": "This is the start of Nest Js.",
        "author": "Me",
        "price": 99,
        "category": "Fantasy",
        "createdAt": "2024-09-03T18:24:28.650Z",
        "updatedAt": "2024-09-05T18:08:06.513Z",
        "__v": 0
    },
]

Post a Book

Use this endpoint to post a new book to the library system.

Example Request:

POST https://library-api-nestjs.vercel.app/book

Example Request:

{
    "title": "Next Js",
    "description": "This is the start of Nest Js.",
    "author": "Me",
    "price": "99",
    "category": "Fantasy"
}

Example Response:

{
    "title": "Aurelia Js",
    "description": "This is the start of Nest Js.",
    "author": "Me",
    "price": 99,
    "category": "Fantasy",
    "_id": "66dad93057b8c16a03d72e30",
    "createdAt": "2024-09-06T10:28:00.771Z",
    "updatedAt": "2024-09-06T10:28:00.771Z",
    "__v": 0
}

Get a Book by ID

Use this endpoint to retrieves details of a specific book by its ID.

Example Request:

GET https://library-api-nestjs.vercel.app/book/book-id

Example Response:

    "title": "Next Js",
    "description": "This is the start of Nest Js.",
    "author": "Me",

    "createdAt": "2024-09-03T18:24:28.650Z",
    "updatedAt": "2024-09-05T18:08:06.513Z",

}

Update a Book

Use this endpoint to update a book in the library system.

Example Request:

PUT https://library-api-nestjs.vercel.app/book

Example Request:

{
    "description": "This is the description.",
    "price": "299"
}

Example Response:

{
    "_id": "book-id",
    "title": "Aurelia",
    "description": "This is the description.",
    "author": "Me",
    "price": 299,
    "category": "Fantasy",
    "createdAt": "2024-09-03T18:24:28.650Z",
    "updatedAt": "2024-09-06T10:51:21.091Z",
    "__v": 0
}
